# Insider Threat Detection Using Behavioral Modeling

## Project Overview
This project aims to detect insider threats by modeling user behavior from enterprise logs using machine learning techniques.

## Objectives
- Model normal user behavior
- Detect anomalous activity
- Reduce false positives
- Provide explainable alerts

## Threat Model
We assume attackers:
- Have legitimate credentials
- Operate within enterprise systems
- Attempt data exfiltration or privilege misuse

## Technologies
- Python
- Scikit-learn
- XGBoost
- TensorFlow / PyTorch
- SHAP (Explainability)

## Evaluation Metrics
- Precision
- Recall
- PR-AUC
- False Positive Rate

## Responsible Use
All datasets used are public or synthetically generated for research purposes only.
